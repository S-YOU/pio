## 0.5.0 (4/14/2014)

### New features
* Added new class `Pio::Hello`.

### Misc

* Added new class alias `Pio::ARP` => `Pio::Arp`.
* Added new class alias `Pio::LLDP` => `Pio::Lldp`.
* Added new class alias `Pio::DHCP` => `Pio::Dhcp`.
